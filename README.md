# 🧪 My Flask Lab

This is a beginner-level project created using **Flask** and **Jinja2** templating to practice web development with Python.

---

## 📁 Project Structure

my-flask-lab/
├── app.py # Main Flask app
├── templates/ # HTML templates (Jinja2)
│ ├── base.html
│ ├── index.html
│ ├── navbar.html
│ ├── blog.html
│ └── contact.html
├── .gitignore # Prevents uploading .venv, cache, etc.
├── README.md # This file
└── .venv/ (ignored) # Virtual environment




---

## 🚀 How to Run This Project

1. **Clone the repository**  
   ```bash
   git clone https://github.com/jswlak/my_flask_lab.git
   cd my-flask-lab

2. **Create a virtual environment**

    python -m venv .venv


3. **Activate the virtual environment**

  Windows

  .venv\Scripts\activate

  macOS/Linux
  source .venv/bin/activate

4. **Install Flask**

pip install flask

5. **Run Flask app**

python app.py

6. **Open in Browser**

Visit 👉 http://127.0.0.1:5000
